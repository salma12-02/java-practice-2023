# java-practice-2023
Java practice programs completed during introductory programming courses in 2023. Includes fundamentals such as loops, arrays, methods, and user input.

This repository contains basic Java programs I wrote while studying Introduction to Programming in 2023.  
The projects cover key programming concepts such as variables, user input, conditional statements, loops, methods, and simple data processing.
## Topics Covered
- Basic input/output
- Currency converter
- Character counter
- Sum and product calculators
- Data entry and simple storage
- Conditional logic (if/else)
- Loops (for, while)
## Notes
These programs reflect my early work in Java and demonstrate foundational programming skills.
